How I would render my metadata:
I would render my metadata by giving each section a clear title or tag. This would help to section the names so that they aren't jumbled together without an indication of why they are there. I would make sure to add in spacing between each item to make it easier to read everything. I would also space it out from the score. I also think it would be good (if possible) to add any links that may be included alongside the metadata alongside the text. This would make it easier for users to access the links.

I would also make sure there is a margin between the side of the page and the text and make the font bigger so that it is easier to read.
